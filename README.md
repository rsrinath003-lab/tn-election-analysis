# Tamil Nadu Election Analysis Dashboard (2001–2021)

This project analyzes Tamil Nadu Assembly elections across five election cycles (2001–2021), covering 234 constituencies.

## Key Insights
- Strong anti-incumbency trend (4 out of 5 elections)
- Significant seat swings between major parties
- Clear geographic voting patterns (Kongu belt vs Delta)
- High constituency-level volatility
